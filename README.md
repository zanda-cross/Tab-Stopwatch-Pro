# Tab-Stopwatch-Pro

A lightweight, floating stopwatch extension for Firefox and Chrome. Designed for workflow tracking with keyboard-first controls.

## 🚀 Installation
Since this is a self-signed extension, follow these steps to install it on Firefox:
]
1. Go to the [Releases](../../releases) page of this repository.
2. Download the [`tab-stopwatch.xpi`](https://github.com/zanda-cross/Tab-Stopwatch-Pro/blob/9709eb022727d960bed065d3e79972aa47cce737/af153b21acd84ef3a91c-1.0.xpi file.)
3. Open Firefox on your work computer.
4. **Drag and Drop** the `.xpi` file into any open Firefox tab.
5. Click **Add** when the permission prompt appears.

## ⌨️ Shortcuts
Once activated via the extension icon:
* **C or V**: Start / Reset the timer.
* **X**: Pause the timer.
* **✕ (on widget)**: Close and remove from page.

## 🛠️ Developer Notes
- **Privacy**: This extension collects zero data. It runs entirely locally.
- **Build**: The code in the root directory is obfuscated for protection. 
- **Source**: Human-readable source code is stored in `Source_Code.zip` for review/audit purposes.
